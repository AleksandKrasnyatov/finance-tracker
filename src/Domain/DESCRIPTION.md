# Основные сущности и их отношения

### Юзер (User)
1. У юзера есть счет или счета

### Счет (Account)
1. У счета есть юзер или юзеры, которые могут что-то делать со счетом
2. У счета есть категории, под которыми добавляются транзакции

### Категория (Category)
1. Категория относится к счету
2. У категории есть создатель — юзер
3. У категории есть транзакции

### Транзакция (Transaction)
1. Транзакция относится к счету
2. У транзакции есть категория
3. У транзакции есть создатель — юзер
